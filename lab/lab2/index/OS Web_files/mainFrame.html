Project Description
Design & Development of a Simulated Unix-based File System.

Your project is to implement a simple file system using C language. You can use any execution platform and compiler to compile and run your program. You will be given code to simulate access to a block-oriented storage device. Your file system will be based on a simulated block-oriented storage device. The interface and functionality of the file system are similar to a simple Unix file system. Your grade will be based on the design of the data structure, correctness of execution and on the quality and style of your code and documentation. More description about the project will be posted here.

  • Full description of the Simple Unix-based File System Project: PDF
    • The required C file sfstest.c for the project. Make sure to read this file, in contains important hints.
    • The required C file blockio.c and its corresponding header file blockio.h for the project.
    • The simulated file system will be implemented using a single ASCII file called simdisk.data.
    • The required makefile to compile your source files.
  • Template for Design Specification of your Simulated File System Project: Here, in MS Word
  • Design of Data Structures for Simulated Unix File System Project. NOTE: this is designed for space efficiency. Your design and implementation should follow the original Unix File System discussed in lectures: PDF
  • Students will work on the project in groups of three members.
  • Two best projects will be asked to present their work in a class session and each group will receive up to two bonus marks in the final mark.
